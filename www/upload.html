Upload Files

Select a file to upload to the server. Supported file types include images, documents, and text files.



Upload Guidelines

  • Maximum file size: 50MB
  • Supported formats: All common file types
  • Files will be stored in the uploads directory
  • Upload progress will be shown during transfer

Test with curl

curl -X POST -F "file=@/path/to/your/file.txt" http://localhost:8080/upload
