WebServ Features

Discover what makes our HTTP server special

Core Features

  • HTTP/1.1 Compliance - Full support for HTTP/1.1 protocol standards
  • Non-blocking I/O - Efficient event-driven architecture using poll()
  • Multiple HTTP Methods - Support for GET, POST, and DELETE operations
  • Static File Serving - Fast and reliable static content delivery
  • Directory Listing - Auto-generated directory indexes with autoindex

Advanced Features

  • CGI Support - Dynamic content generation with Python and PHP
  • File Uploads - Secure file upload handling with size limits
  • Custom Error Pages - Beautiful, branded error pages for 403, 404, 500
  • URL Redirection - HTTP redirect support for URL rewriting
  • Configuration System - Flexible server configuration with multiple locations

Performance Features

  • Connection Pooling - Efficient connection management
  • Memory Optimization - Minimal memory footprint
  • Fast Response Times - Optimized for low latency
  • Concurrent Connections - Handle multiple simultaneous requests
