Welcome to my WebServ!

This is a simple HTTP server implemented in C++98.

Features

  • HTTP/1.1 compliant
  • Non-blocking I/O with poll()
  • Support for GET, POST, and DELETE methods
  • Static file serving
  • Directory listing
  • CGI support (PHP, Python)
  • File uploads
  • Custom error pages
